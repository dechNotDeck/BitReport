Get BitReport

Bitreport is a ticket system that allows players to communicate problems to the administration team whether they are online or not without the need for an external communication channel.

For a list of commands and permission nodes, check the included plugin.yml file.

Features

  • Creating and claiming tickets
  • Location saving on ticket creation
  • Listing and displaying information about tickets
  • Can retrieve the first available ticket marked as "available"
  • Teleporting to the location the ticket was created via command
  • Ticket resolving, allows the person handling the ticket to leave a note
  • Status messages informing players when their tickets have been claimed and resolved
  • Multiworld support
  • PEX Permission support
